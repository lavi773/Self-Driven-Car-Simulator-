    How Self-Driving Works

    This simulator combines rule-based JavaScript motion planning with a lightweight Flask API that emulates AI decisions. Sensors estimate distance to obstacles, traffic signal status, and lane safety. The car then decides whether to maintain speed, brake, or switch lanes.

    • Perception: Detect vehicles, pedestrians, and static obstacles.
    • Prediction: Estimate how nearby objects will move over time.
    • Planning: Choose safe actions such as stopping at red lights.
    • Control: Smoothly steer and adjust speed to execute actions.

    About Project

    Built as a mini autonomous driving simulator suitable for a BCA Data Science final year project, this web app demonstrates human-readable AI logic, UI animation, and data-driven feedback.
